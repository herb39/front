문자 함수 연습

    0         1         2         3
    0123456789012345678901234567890
    ABCDEFGHIJKLMNOPQRSTUVWXYZabcde
    ABC   BCDEF  GHI  JKLMN... ABC.. OPQ

length : 객체(문자(열))의 길이

substring(a, b) : 문자열 발췌(a부터 b앞까지)

slice(a, b) : 문자열 발췌(a부터 b앞까지)

substr()

replace(a, b) : a를 b로 치환

replaceAll(a, b) : 모든 a를 b로 치환

toUpperCase() : 대문자로 변환

toLowerCase() : 소문자로 변환

concat(a, b) : a와 b문자열 결합

trim() : 문자열의 앞뒤(양쪽)공백을 무시

trimStart() : 문자열의 앞쪽공백을 무시

trimEnd() : 문자열의 뒤쪽공백을 무시

repalceAll(' ', '') : 문자열 안의 모든 공백 무시

charAt(index) : 하나의 문자만 발췌

charCodeAt(index) : index의 아스키코드

split() : 특정 문자를 기준으로 자르고, 잘라낸 값을 배열변수에 담음

indexOf() : 문자(열) 검색시 사용 : 있으면 index 값 / 없으면 -1 반환

lastIndexOf() : 문자(열) 검색시 사용(뒤에서부터 검색) : 있으면 index 값 / 없으면 -1 반환

search() : 문자(열) 검색시 사용 : 있으면 index 값 / 없으면 -1 반환

match() : 문자열 일치 여부 : 있으면 문자열 그대로 반환 / 없으면 공백

includes() : 문자열 포함 여부 : 있으면 true / 없으면 false
