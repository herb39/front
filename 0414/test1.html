정규식 Regular Expression

문자열 : 

    형식 : / ~~~~~ / 플러그인
    플러그인 :
      g(global) : 전체범위 처리
      m(multiline) : 여러줄 처리(textarea 태그)
      s(single line) : 한줄 처리(기본값)
      u(unicode) : 유니코드 처리
      i : 영문자 대문자 소문자 구별x

    정규식을 실행하기 위해서 사용되는 함수
      1. 사용자입력문자열.match(정규표현식변수)   : 포함 유무 비교시 사용
      2. 정규표현식변수.test(사용자입력문자열)    : 구체적인 문자(열) 비교시 사용

form 안의 값을 입력 후 서버로 전송할 때, 유효성 검사
