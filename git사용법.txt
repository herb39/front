* git 등록하기
1) 유저 등록하기
$ git config --global user.name "herb39"

2) 유저의 이메일주소 등록하기
$ git config --global user.email "herb_39@nate.com"

3) 등록 확인
$ git config --list

4) 명령어
git init

git add .

-- git status    // 현재 상태보기

git commit -m "first commit"

git remote add origin https://github.com/cheonbosik/front.git

-- git remote -v   // 잘 연결되었는지 확인...

git push origin master

========================

앞으로  현재 컴퓨터에서 작업할때는?

* 작업한 파일을 다른곳에서 다운받아서 처리할 경우..

-(1)- 집에서 git설치후 환경설정(user명과 메일주소등록)하고, git을 실행후 커맨드모드에서 아래와 같이 2줄을 입력처리한다.

-(2)- 작업폴더 만든후 vs code에서 작업폴더를 연결한다.(작업폴더를 test로 만들었다면, vs code에서 작업폴더를 연결후 터미널에서 git bash로 연결해서 아래명령을 차례로 수행한다.
